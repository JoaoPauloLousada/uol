# Word2vec - model answer
1 - Unlike FreqDist or TF-IDF, which focus on raw word counts or frequencies, Word2Vec captures relationships between words (e.g. 'king' relates to 'queen' as 'man' relates to 'woman') by embedding them in a dense vector space.

2 - Word2Vec can infer relationships for unseen words using pre-trained embeddings, which is not possible with static frequency-based approaches.
